# Student Grade Calculator

## Description

The Student Grade Calculator is a simple Python project that calculates the total marks, average marks, and grade of a student based on the marks obtained in five subjects. This project is designed for beginners to understand basic Python concepts such as variables, user input, arithmetic operations, and conditional statements.

## Objective

To develop a Python program that:

* Accepts student details and marks.
* Calculates the total and average marks.
* Determines the student's grade.
* Displays a student report.

## Features

* Takes Student Name and Roll Number as input.
* Accepts marks for five subjects.
* Calculates total marks.
* Calculates average marks.
* Assigns grades based on the average.
* Displays the complete student report.

## Grade Criteria

| Average Marks | Grade |
| ------------- | ----- |
| 90–100        | A     |
| 75–89         | B     |
| 60–74         | C     |
| 35–59         | D     |
| Below 35      | F     |

## Technologies Used

* Python 3
* IDLE Python

## How to Run

1. Install Python 3 on your computer.
2. Open IDLE Python.
3. Create a new file named `student_grade_calculator.py`.
4. Copy and paste the Python code into the file.
5. Save the file.
6. Press `F5` or select **Run → Run Module**.
7. Enter the student details and marks when prompted.
